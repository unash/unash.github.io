☆实现功能

在IPv6的网络环境下，可以访问万维网。可以访问国内的，而且具有翻*墙的效果。教程稍有复杂，但有些步骤读者可能已经设置好了，通篇读写来之后能够对整个设置有一个比较清晰的了解。Here we go！

☆网络环境要求

具有IPv6的网络环境，一般高校校园网的网络是ipv6与ipv4混合的（如果对各个高校IPv地址感兴趣的话，可以访问这个列表）。然而即使是IPv6网络，当使用了路由之后，有可能会变成ipv4网。在将学校的网络接口接入到路由器之后，路由器通常有两种设置模式，一种是交换机模式，一种是路由模式。交换机模式下，不改变网络的类，即原来是ipv6的经过交换机之后依然是ipv6；但是在路由模式下，经过之后则会变成ipv4模式，这样就无法使用ipv6了。
然而路由器的默认设置是路由模式，那么可以设置为交换机模式。以市场上销售的那种百元左右的小型路由器为例，通常的做法是，不使用路由器上的WAN端口，只使用剩下的LAN端口连接所有的网线。即所有的网线都接在LAN上，重启一些就可以了。

☆IPV6的安装

具有了ipv6的网络之后，还需要电脑支持ipv6网络。windows 7，已经默认安装了IPV6协议。但是对于windows XP，需要先安装IPV6，可搜索一下如何安装。安装成功之后，在cmd命令行中，键入ipconfig -all，会出现一串的内容，其中会出现ipv6相关的内容，比如类似2001:0:9d38:6ab8:18d8:26df:5d96:f4df类似的地址。OK，IPv6访问无碍，可以进行主要的设置步骤了。

☆host文件的使用

ipv6网络安装之后，为了能否访问国内外的ipv6网络（其实主要是国外的，国内的比如ipv6.baidu.com虽然可以打开首页，但是也只是首页而已，其他的页面是无法通过ipv6访问的），需要修改host文件，下载地址见括号（本地下载，去掉txt后缀名；官方下载）。复制到目录C:\Windows\System32\drivers\etc中，替换掉原来的hosts文件。之后，在浏览器中输入www.google.com，就可以访问了。
如果不出大的问题的话，也可以访问维基百科www.wikipedia.org，包括google的其他的大部分服务也可以访问。

☆代理的设置

完成上面的步骤之后，进度已经完成了30%了，下一步需要设置访问代理。我们知道，由于GFW的存在，国外的很多网站无法访问，由此催生了代理方法的发展，其中一个比较知名的是goagent，架设在google的Google AppEngine上。更多相关内容，可以去goagent的官方网站看看。

要从头开始使用goagent，首先需要申请一个google账户，之后进入https://appengine.google.com/? ，开通google appengine的服务，新建一个应用。


创建成功之后，记住application的名字，然后下载GoAgent官方服务器文件，解压；如果这个网址打不开，那么使用百度网盘下载，或者使用谷歌网盘下载。解压之后上传的过程参考GoAgent官方中的文档，此处不再赘述。如果上传成功，在使用local文件夹中的程序的时候，需要修改一下proxy.ini文件，将[gae]的这一段修改为如下：
[gae] appid = XXXXX _#这里XXXXX更换为你自己的Google AppEngine的application名字_ password = path = /2 profile = google_ipv6 crlf = 1 window = 4
然后用管理员身份启动goagent.exe程序，在得到如图所示的效果之后，基本就可以了：

☆Goagent的使用

启动之后，还需要修改一下浏览器的代理设置，才能够使用。通常设置的地方在Internet选项->连接->本地连接，设置代理的地址为127.0.0.1，端口为8087（这个端口可以修改，在proxy.ini文件中）。确定之后，就可以使用浏览器访问万维网了。

☆Proxy SwitchSharp的使用

但是这样设置之后，如果没有启动代理，在希望不通过代理访问网络的时候，还需要手动取消代理，比较麻烦，如果是chrome浏览器的话，这样的问题就可以大大化简了。下载Google chrome proxy switchysharp，本地下载安装方式（右键另存为此文件，修改后缀名为.crx，安装到chrome（如果不知道怎么安装，就直接把这个文件拖到google浏览器中，直接安装已经行不通来，需要通过工具->扩展程序进行安装），或者使用谷歌网盘下载并直接安装。安装成果之后在浏览器窗口的右上角会出现图标，点击，进入设置。读者估计自己看看就学会了，不赘述。通过插件SwitchyProxy,可以比较灵活地切换代理。

如果懒得配置了，可以下载我这个witchyProxy的备份文件，直接还原就可以了。

☆其他IPv6访问方式

如果只是临时使用网络而懒得再去设置代理，浮茗在此推荐一个代理网站，可以直接通过修改url访问外网。

访问网址为：http://www.sixxs.net/tools/gateway/

在最后的那个框中输入你需要访问的地址，然后点击Browse over IPv6，就可以直接访问了。

☆注意事项

由于Goagent经常被屏蔽之后就不能用了，所以如果某日不可以通过代理上网了，则不妨下载一个新的goagent服务器文件重新上传使用。
google的application有流量限制，每天不可超过1G，所以用代理上网时，每天尽量控制在1G之内。

无论使用的什么代理，如果对代理的原理不清楚，请勿通过代理进行一些涉及财务的操作，以防密码泄露。
