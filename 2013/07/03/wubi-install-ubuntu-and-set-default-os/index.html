wubi安装ubuntu,设置默认操作系统

By 浮茗

Jul 3 2013 Updated:Jan 31 2015

Contents

听说ubuntu在13.04中不再只是wubi安装ubuntu来，只好下载了一个ubuntu12.04的系统进行wubi安装。安装完成之后，开机之后可以选择是使用windows还是使用ubuntu，界面如下：

默认等待时间是10s，默认选项是windows7，选择了ubuntu之后，还要选择是正常启动ubuntu还是windows7，界面如下（从网上找的图，不一定完全准确，大概样子是这样）：

既然安装了ubuntu，主要还是为了转移到Linux阵地，所以希望默认启动ubuntu系统，而且不想等10s这么长的时间。搜索了一下，发现大部分的教程（利用linux的grub启动工具）都是修改的第二个图，而实际上，我希望修改第一个图中的启动顺序。后来发现，由于是“windows启动管理器”，所以还应该在windows中修改（毕竟利用wubi安装ubuntu，ubuntu只是作为windows的一个应用程序存在的）。方法如下：计算机->属性->高级系统设置->启动和故障恢复->设置->默认操作系统。

到这一步为止，就知道应该怎样设置了。当然如果希望修改ubuntu的启动顺序，可在进入ubuntu之后，修改/boot/grub/grub.cfg文件，具体略过。
